# API Access

There are a few supported APIs to access ZORA chain information.

1. RPC Access:

    Use viem/ethers with a public RPC url or [your own node](https://github.com/conduitxyz/node) to access information.


2. Explorer Access:

    Use blockscout APIs to access data:

    * [rest endpoint](https://explorer.zora.energy/api-docs) / [docs](https://docs.blockscout.com/for-users/api/rest-api-endpoints)
    * [graphql endpoint](https://explorer.zora.energy/graphiql) / [docs](https://docs.blockscout.com/for-users/api/graphql) 
    * [rest / etherscan type api endpoint](https://explorer.zora.energy/api) / [docs](https://docs.blockscout.com/for-users/api/rpc-endpoints)


3. ZORA NFT API:

    Use [api.zora.co](https://api.zora.co/) with ZORA as the network parameter to query NFT information on ZORA network.
